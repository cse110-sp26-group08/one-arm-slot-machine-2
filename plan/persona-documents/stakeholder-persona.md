# Stakeholder Persona

### Name: Joe

### Description:

Joe is a VC in an up and coming fund. He likes games that keep users engaged. He wants to make it big in the industry so he cares about how they are reviewed on the app store. Likes vibrant and colorful UI because they have found this works well in practice for keeping users in the game.

### User Stories:

#### Basic Needs

- [x] As a stakeholder who cares about the overall UI of the app, I would like a vibrant and colorful UI and a fitting music playlist along with it so that users will keep playing
- [x] As a stakeholder who cares about the long term success of the game, I would want periodic updates so that Users keep playing the game and the game does not die.
- [x] As a stakeholder who wants maximum number of users, I want the game to be responsive to different screen sizes and accessible, so that I can attract the most number of users.

#### Additional Features

- [x] As a stakeholder who cares about the reception of the game, I would want users to not lose motivation to play by losing continuously, so that they can write a nice review and my reputation goes up.
- [ ] As a stakeholder who does not want his game to die, I want there to be a streak feature (daily spin), so that my users keep on playing the game everyday.
- [ ] As a stakeholder, I want new slot machines to be unlocked after a certain streak threshold is reached, so that users have a reason to maintain a streak.
